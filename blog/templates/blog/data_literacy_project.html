WHAT WE DO?

Upskilling healthcare professionals to be data-literate

What do data-literate professionals do? They read, create, understand, summarize , and communicate data as information Accelerate decision-making, tell action-inspiring, life-changing stories from Data.

WHAT IS DATA LITERACY?

Comprehend data and use it to make decisions with confidence

Data can help professionals with critical information as they make decisions. Ultimately, data literacy provide access to information that supports better decision making.

LEARN IN DETAILS

Understand Data Sources

Learn the right tools and techniques to access and request data from multiple sources. Either be the web, databases or from a real survey.

Read,Write & Summarise Data

Be able to clean, simplify, consolidate and consume data. Build healthy, positive and productive relationship with data.

Communicate Data

Be able to deliver your insights, findings and approaches to the audience in best way possible through the power of Data Visualization

Describe the Data

Be data brilliant. Gather insights, accelerate decision-making, tell action-inspiring, life-changing, persuasive stories from Data.

StoneAgeEntrepreneur: The Data Litracy project

What will you learn from The Data Literacy Project?

Designed to provide best hands-on learning experience by engaging in direct application of skills, theories and models. Our courses will train you to be able to apply the knowledge you have gained to real-world settings.

LEARN IN DETAILS

Statistics & Probability: From the grounds up

From Measure of Central Tendency, to Probabilistic Theories; we have got you covered. What is Qualitative and Qualitative Data?

Python Programming Language

Learn to use Python libraries like Pandas, Numpy, Matplotlib, Scipy to read, store, analyse and visualize data.

Exploratory Data Analysis

Learn to identify variables, discover patterns, handle missing values & verify assumptions required for model fitting and hypothesis testing.

Learn from Data

Here, we will teach you the theoretical understanding of basic statistical models like linear regression and classification.

How our courses are designed:

How you will learn

Live and Recorded Lecture Videos

Courses run over 2-2.5 months, 3 days a week with a 2-hour video lecture that you can watch live or recorded.

Real-World Projects

You'll get to practice and improve your skills with weekly assignments, and build a portfolio project using real-world data.

Interactive Notebooks

Start instantly. Courses are taught using interactive Jupyter notebooks which you can execute and experiment with on our platform

Community Driven

Ask questions, get help, share your work and learn together with community of like-minded learners.
